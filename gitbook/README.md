# Микроконтролеры от команды [ант-лаб](https://ant-lab.ru/)

### Инструкция

1. Установить [Obsidian](https://obsidian.md/)
2. Клонировать этот репозиторий
3. В Obsidian:
    - Открыть локальную копию этого репозитория
    - Открыть заметку **Микроконтроллеры**
4. Проходить **темы**:
    - читать вступление
    - пользоваться материалами
    - выполнять задания
    - контролировать понимание темы контрольными вопросами
