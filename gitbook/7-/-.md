# Задания

1. Создать новый проект прошивки реализующей портативный измеритель состояния атмосферы
2. Добавить в проект существующую библиотеку датчика, настроить телеметрию измерений измерения
3. Добавить в проект существующую библиотеку дисплея, вывести на дисплей пробный текст
4. Разработать GUI прибора в виде текста с измерениями на дисплее
5. Оживить GUI, заменив заглушки измерений на результаты измерений
6. Реализовать журнал измерений
7. Реализовать API прибора с заданием периода записи измерний в журнал, возможнсотью скачать журнал и возможностью задать текущее время
8. Запустить прибор в работу на 5 минут, скачать журнал измерений
9. Построить график измерений Python-скриптом
10. Добавить в прошивку (прикладной уровень прибора) возмжность отображать график или графики измерений

# Не забыть

1. Запушить исходники новой прошивки, Python-интерфейс и скрипты на GitHub
