# Задания

1. Создать новый проект прошивки, отображающей изображения на дисплее
2. Создать библиотеку для дисплея
3. Создать драйвер дисплея, заполняющий его заданным цветом
4. Реализовать градиентную заливку экрана
5. Реализовать отображение цветного прямоугольника на заданном фоне
6. Реализовать выбор ориентации изображения
7. Реализовать задаваемую частоту обновления изображения
8. Реализовать N двигающийся по экрану, меняющих цвет прямоугольников
9. Реализовать отображение скомпилированной картинки
10. Реализовать Python-скрипт загрузки картинки на экран

# Не забыть

1. Запушить исходники новой прошивки, Python-интерфейс и скрипты на GitHub
