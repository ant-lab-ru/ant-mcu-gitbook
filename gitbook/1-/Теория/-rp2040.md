| Адрес       | Название                          | Пояснение                                                                                                               |
| ----------- | --------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| 0x000       | RP2040 Boot Loader (.boot_loader) | Второй этап загрузчика (boot2) во внешней Flash: инициализирует QSPI/XIP и передаёт управление основному коду прошивки. |
| 0x100       | Initial Stack Address             | Начальное значение указателя стека (SP), которое ядро загружает при старте.                                             |
| 0x104       | Reset Handler                     | Точка входа после сброса: стартовый код (инициализация), затем переход в main.                                          |
| 0x108       | NMI Handler                       | Обработчик немаскируемого прерывания (NMI).                                                                             |
| 0x10C       | HardFault Handler                 | Обработчик “тяжёлой ошибки” (HardFault): вызывается при критических исключениях.                                        |
| 0x12C       | SVC Handler                       | Обработчик SVC (Supervisor Call) — программного прерывания/системного вызова.                                           |
| 0x138       | PendSV                            | Обработчик PendSV — часто используется планировщиком в RTOS для переключения задач.                                     |
| 0x13C       | SysTick Handler                   | Обработчик системного таймера SysTick (периодический “тик”).                                                            |
| 0x140       | ISR 0                             | Адрес обработчика внешнего прерывания IRQ0.                                                                             |
| 0x144       | ISR 1                             | Адрес обработчика внешнего прерывания IRQ1.                                                                             |
| …           | …                                 | Остальные записи таблицы векторов для IRQ2…IRQ30.                                                                       |
| 0x1BC       | ISR 31                            | Адрес обработчика внешнего прерывания IRQ31.                                                                            |
| 0x1C0       | Code (.text)                      | Машинный код программы (инструкции), который исполняет процессор.                                                       |
| после .text | Data (.rodata & .data)            | .rodata — константы/строки (только чтение). .data — начальные значения переменных, которые при старте копируются в RAM. |
