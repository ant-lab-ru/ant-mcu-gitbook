Вы научились создавать и записывать прошивку. Но что делать, если нужна дополнительная информация? Где искать характеристики GPIO? Как найти пример работы с UART?

**Главный принцип:** Документация — это не книга для чтения от корки до корки. Это справочник, в котором вы ищете конкретную информацию.

### Типы документации

**1. Datasheet (даташит) — технический паспорт**

**Что внутри:**
- Характеристики микроконтроллера
- Распиновка (какой вывод за что отвечает)
- Электрические параметры (напряжение, ток, температура)
- Описание периферии

**Когда использовать:**
- Нужны характеристики (сколько GPIO? сколько тока выдаёт вывод?)
- Нужна распиновка (какой пин подключить?)
- Нужны электрические параметры (можно ли подключить 5V устройство?)

**Для RP2040:**
- [RP2040 Datasheet (PDF)](https://datasheets.raspberrypi.com/rp2040/rp2040-datasheet.pdf) — документация на микроконтроллер RP2040
- [Pico Datasheet (PDF)](https://datasheets.raspberrypi.com/pico/pico-datasheet.pdf) — документация на плату RaspberryPi Pico

**Как искать в datasheet:**
1. Откройте PDF
2. Нажмите Ctrl+F (поиск)
3. Введите ключевое слово (например, "GPIO electrical")
4. Читайте только найденный раздел

**Пример:**
```
Вопрос: Сколько тока может выдать GPIO в RP2040?
Действия: Ctrl+F → "GPIO electrical" или "output current"
Ответ: 4-12 мА (в зависимости от настройки drive strength)
       Drive strength 0: ~4 мА
       Drive strength 3: ~12 мА
```

**2. SDK Documentation — описание функций**

**Что внутри:**
- Описание всех функций SDK
- Параметры функций
- Примеры использования

**Когда использовать:**
- Нужно понять, как использовать функцию
- Нужно узнать, какие параметры принимает функция

**Для Pico SDK:**
- [Pico SDK Documentation (Online)](https://raspberrypi.github.io/pico-sdk-doxygen/)

**Пример:**
```
Вопрос: Как использовать функцию gpio_init()?
Действия:
1. Открыть SDK Documentation
2. Найти раздел "GPIO"
3. Найти функцию gpio_init()
4. Прочитать описание

Ответ:
void gpio_init(uint gpio)
- gpio: номер GPIO (0-29)
- Инициализирует GPIO для использования
```

**3. Примеры кода (Examples)**

**Что внутри:**
- Готовые рабочие примеры
- Код для GPIO, UART, SPI, I2C и т.д.

**Когда использовать:**
- Нужен рабочий пример для старта
- Нужно понять, как использовать периферию

**Для Pico:**
- [Pico Examples (GitHub)](https://github.com/raspberrypi/pico-examples)
- Или локально: `pico-sdk/examples/`

**Структура:**
```
pico-examples/
├── blink/          ← Мигание светодиода
├── uart/           ← Примеры UART
│   └── hello_uart/ ← Простой вывод
├── spi/            ← Примеры SPI
├── i2c/            ← Примеры I2C
└── adc/            ← Примеры АЦП
```
