# Результат

- [ ] Работает ssh соединение с github
- [ ] На компьютере в папке `C:/Repositories/pico/` лежит склонированный репозиторий `mcu`
- [ ] Структура папок как на схеме ниже:

```
Repositories
|-- pico
	|-- pico-sdk
	|-- mcu
```
