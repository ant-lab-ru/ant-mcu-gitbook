> [!INFO] Факт
> В операционных системах Linux и MacOS при установке программ через терминал они скачиваются из интернета из специальных репозиториев при помощи [[Менеджер пакетов|пакетных менеджеров]].

> [!ATTENTION] Внимание!
> Для наших задач подойдёт любая версии **git**. Установленная на вашем компьютере версия **может** отличаться.

# Linux

1. В терминале выполните [[обновление списка доступных пакетов]]:

``` bash
sudo apt update
```

2. В терминале установите программу **git**

``` bash
sudo apt install 
```

3. Запросите версию программы **git**

``` bash
git --version 
```

4. Убедитесь в том, что программа **git** сообщает свою версию:

``` bash
git version 2.42.0
```

# MacOS

1. В терминале при помощи программы [[Менеджер пакетов для macOS|brew]] установите программу **git**:

``` bash
brew install git
```

2. Запросите версию программы **git**

``` bash
git --version
```

3. Убедитесь в том, что программа **git** сообщает свою версию:

``` bash
ivanpotylitcyn@Ivans-MacBook-Pro ~ % git --version
git version 2.50.1 (Apple Git-155)
ivanpotylitcyn@Ivans-MacBook-Pro ~ %
```

# Windows

1. Cкачайте установщик программы **git**:

https://git-scm.com/install/windows

2. Запустите установщик
	- При установке обязательно поставьте галочку **Добавить git в Path**

3. В терминале убедитесь в том, что программа **git** сообщает свою версию:

``` bash
PC C:\users\potylitcyn> git --version
git version 2.42.0.windows.2
PC C:\users\potylitcyn> 
```
