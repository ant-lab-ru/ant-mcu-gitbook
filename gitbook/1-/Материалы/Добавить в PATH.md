> [!HINT] Для справки
> [[Переменная среды|Переменные среды]] можно изменить временно (на текущую сессию терминала) и постоянно. В этой инструкции написано как добавить путь в PATH на постоянной основе.

> [!ERROR] Обратите внимание
> Измененное значение переменной среды применяется только после перезапуска терминала.

# Windows

1. Открываем системное окно поиск и пишем: `изменение системных переменных среды`. Выбираем найденный вариант:

![[Pasted image 20260126165218.png|400]]

2. В открывшемся окне `Свойства системы` выбираем снизу справа переменные среды:

![[Pasted image 20260126165255.png|400]]

3. В открывшемся окне `Переменные среды`, в нижней половине окна, в `Системные переменные` выбираем переменную `Path` или `PATH` (может называться и так и так) и нажимаем изменить:

![[Pasted image 20260205195049.png|400]]

4. В открывшемся окне `Изменить переменную среды` нажимаем кнопку `Создать`, внизу добавляется новая строчка, куда нужно вписать путь к нужному исполняемому файлу. Путь должен содержать обратные слэши: `\`:

![[Pasted image 20260126165827.png|400]]

![[photo_2026-01-26_17-32-10.jpg|400]]

5. Нажимаем во всех открытых окнах `ОК`. Готово!

# MacOS

На MacOS добавление в path должен выполнять пакетный менеджер brew, поэтому добавление в path вручную не требуется. Если это все-таки нужно, то:

1. Открываем терминал и открываем в редакторе nano конфигурационный файл `~/.zshrc`:
		`nano ~/.zshrc`
2. Добавляем в конец файла строку с нужным путем:
		export PATH="$PATH:/usr/local/myapp/bin"
3. Сохраняем файл:
		Ctrl+O -> Enter -> Ctrl+X
