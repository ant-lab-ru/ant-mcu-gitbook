> [!INFO] Определение
> 
> **Homebrew (или просто «brew»)** — это сторонний менеджер пакетов для macOS, упрощающий установку, обновление и удаление программного обеспечения через командную строку.

# Ключевые особенности

1. **Простота использования**. Позволяет устанавливать программы одной командой вида `brew install имя_пакета`.
2. **Отсутствие необходимости в `sudo`**. Работает без прав суперпользователя, что повышает безопасность.
3. **Управление зависимостями**. Автоматически устанавливает необходимые библиотеки и компоненты.
4. **Чистый деинсталл**. Полностью удаляет пакеты без остаточных файлов.
5. **Расширяемость**. Поддерживает пользовательские репозитории («taps») и собственные формулы пакетов.

# Основные компоненты

- **Formulae (формулы)** — скрипты на Ruby, описывающие процесс сборки и установки консольных программ.
- **Casks** — механизм для установки GUI‑приложений (например, Firefox, VS Code).
- **Taps** — сторонние репозитории с дополнительными пакетами.
- **Bottles** — предварительно собранные бинарные пакеты для ускоренной установки.

# Как работает

1. Пакеты устанавливаются в каталог `/opt/homebrew` (на Apple Silicon) или `/usr/local` (на Intel).
2. Homebrew создаёт символические ссылки (`symlinks`) в стандартные системные директории.
3. Зависимости изолируются, чтобы избежать конфликтов с системными компонентами.

### Примеры команд

- Поиск пакета: `brew search название`
- Установка: `brew install git`
- Обновление: `brew update && brew upgrade`
- Удаление: `brew uninstall python`
- Информация о пакете: `brew info node`

# Почему это полезно

- **Для разработчиков**: быстрая установка инструментов (Python, Node.js, Git и др.).
- **Для обычных пользователей**: альтернатива ручному скачиванию `.dmg`-файлов.
- **Экономия времени**: автоматизация рутинных операций по сборке ПО.

# Установка

Требуется только **Command Line Tools for Xcode** (можно установить через `xcode-select --install`). Сама установка Homebrew выполняется одной командой:

``` bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```


> [!NOTE] Примечание
> 
> Homebrew — это открытый проект (лицензия BSD), поддерживаемый сообществом. Он стал де‑факто стандартом для управления ПО в macOS, заполняя пробел, оставленный отсутствием встроенного пакетного менеджера от Apple.
