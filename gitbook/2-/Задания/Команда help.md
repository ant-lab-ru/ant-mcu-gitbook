# Результат

- [ ] в протокол добавлена команда help, которая выдает список команд, доступных в устройстве;
- [ ] для каждой команды отображается ее имя и описание;
- [ ] прошивка скомпилирована и запущена;
- [ ] на отправку команды `help` через COM-порт, после нажатия `Enter`, устройство выдает список всех команд с описаниями
![[ПримерHelp.PNG]]

   >[!ATTENTION] Обратите внимание
   >Вывод описания команд должен быть реализован с помощью поля ```command_help``` структуры ```api_t```
   
   > [!ATTENTION] Подсказка
> Для того чтобы не возникало ошибок с порядком объявления функции-callback и массивом структур  ```api_t``` используйте **прототип функции**.
