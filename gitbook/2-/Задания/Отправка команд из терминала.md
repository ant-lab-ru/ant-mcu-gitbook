# Инструкция

> [!ATTENTION] Внимание!
> В этом задании вы должны продолжать работать с проектом `02-echo`.

1. После вывода кода символа в ```ASCII```, добавить логику включения светодиода при приеме символа `e` (`e` от "enable"):

```
switch(symbol)
{
case 'e':
	gpio_put(LED_PIN, true);
	printf("led enable done\n");
	break;

default:
	break;
}
```

2. добавить логику выключения светодиода при приеме символа `d` (`d` от "disable");
3. добавить логику вывода имени и версии устройства при приеме символа `v` (`v` от "version");
4. cобрать прошивку, загрузить в устройство, запустить и проверить отклик на отправку символов через терминал, включение выключение светодиода, вывод версии.
