# Прошивка

- Подключить библиотеку для работы с вводом/выводом данных (Standard Input/Output)  и библиотеку с базовыми функциями работы с микроконтроллером 

1. подключить заголовочные файлы стандартной библиотеки ввода вывода и  библиотеки базового взаимодействия с микрокнтроллером;
2. инициализировать систему ввода/вывода `stdio`;
3. добавить в тело бесконечного цикла блокирующее посимвольное считывание из USB через stdio:

``` c
char symbol = getchar();
```

4. добавить вывод в терминал считанный символ и его код в ```ASCII```. Пример для вывода кода символа в ```ASCII```:

``` c
printf("received char: %c [ ASCII code: %d ]\n", symbol, symbol);
```

5. cобрать прошивку, загрузить в устройство, запустить и проверить отклик на отправку символов через терминал.
