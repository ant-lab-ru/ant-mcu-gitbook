
# Windows

Определить имя COM-порта:
- Нажмите `Win + X` и выберите «Диспетчер устройств».
- Разверните список **Порты (COM и LPT)**.

![[Pasted image 20260204105514.png]]

- Устройство будет отображаться как «Название устройства (COM#)», например, `Устройство с последовательным интерфейсом USB (COM13)`.

# MacOS / Linux

1. введите в терминале:

``` bash
ls /dev
```

2. в выводе найдите устройство, содержащее `tty.usbmodem`. Скопируйте его имя.
