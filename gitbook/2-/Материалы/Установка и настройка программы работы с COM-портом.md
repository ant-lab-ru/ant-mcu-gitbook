# Windows

1. Скачать бинарный файл консольной программы [PUTTY](https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html):

![[Pasted image 20260204145655.png]]

2. Запустить скаченный файл. Основное окно программы PuTTY выглядит следующим образом:

![[Pasted image 20260202105738.png]]

3. После того, как вы загрузите рабочую прошивку с USB CDC в Pico, [[определить имя COM-порта]];
4. Теперь можно приступить к настройке COM-порта. В `Connection type выберите `Serial. В `Serial line` укажите название COM-порта:

> [!ATTENTION] Внимание!
> В данном случае мы подключаемся не напрямую к UART, а к виртуальному COM-порту, который создает микроконтроллер RP2040 на PICO. USB данные не с фиксированной скоростью (baudrate), а пакетами по высокоскоростному USB-протоколу. **Поэтому в этом менять значение скорости не обязательно.**

![[Pasted image 20260204110523.png]]

# MacOS / Linux

1. [[Определить имя COM-порта]];
2. запустите встроенное приложение `screen`. Пример запуска:

``` bash
screen /dev/tty.usbmodem1101
```
